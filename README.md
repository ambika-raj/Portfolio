# Personal Portfolio Website

This is my personal portfolio website created as part of my academic mini project.  
The website showcases my basic information, skills, projects, and a contact form.  
It is designed to be simple, responsive, and easy to understand, using only HTML and CSS.


## Tech Stack

- HTML5  
- CSS3  
- Flexbox & Grid  
- CSS Animations  
- Media Queries
